Introducing scrollit.js

scrollit.js is a very simple jquery script that smoothly scrolls to a target. It increases user experience and makes your website a little bit nicer. Beside jquey you additionaly need jquery-ui for easing effects. scrollit.js is easy to use and works allmost out of the box. See the howto for details.

How to integrate scrollit.js

          function scrollIt(element, easing, offset, callback)
          {
            $('html, body').animate({ scrollTop: ($(element).offset().top) + offset}, 1500, easing);
            
            if(callback && (typeof(callback) === "function") )
              {
                callback();
              }
          }
